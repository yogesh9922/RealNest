🚀 Overview
RealNest is a full-featured real estate web platform that allows users to browse, list, and purchase properties. It integrates a machine learning-based price prediction system and a real-time chatbot, enhancing the buying experience with intelligent assistance and scheme eligibility suggestions.

🔧 Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: PHP, MySQL
Machine Learning: Python, Random Forest
Other Features: Chatbot, Government Scheme Eligibility Checker
✨ Features
🏘️ Property listings with categories like Apartment, Villa, Shop, etc.
📊 Price prediction using Random Forest ML algorithm
🤖 Real-time chatbot for user support
🏛 Government scheme eligibility suggestions based on user data
🔐 User login & authentication
📋 Property submission for sell/rent
📦 Data management using MySQL
💬 Live chat for buyer-seller communication
